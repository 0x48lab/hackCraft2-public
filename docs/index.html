HackCraft2

Minecraftの新しいModdingプラットフォーム

主な機能

ビジュアルプログラミング

Scratchベースのビジュアルプログラミング環境で、直感的にModを作成できます。

3Dプレビュー

作成したModの3Dモデルをリアルタイムでプレビューできます。

コードエディタ

高度な機能が必要な場合は、コードエディタで直接プログラミングも可能です。

ダウンロード

最新リリース

最新の安定版ビルドをダウンロードできます。

始め方

インストール方法

  1. 最新のリリースをダウンロード
  2. Minecraftのmodsフォルダにjarファイルを配置
  3. Minecraftを起動

開発環境のセットアップ

  1. リポジトリをクローン
  2. Gradleを使用してビルド
  3. 開発用の環境をセットアップ
